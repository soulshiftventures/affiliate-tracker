**Total Time: 5 minutes per day**

```bash
cd /Users/krissanders/DeepDiveSystems/projects/affiliate_swarm
python3 tracker.py
```

Then open your browser to:
```
http://localhost:9000/api/stats
```

Or view the visual dashboard (if running):
```
open dashboard.html
```

---

## Step 2: Look at the Big Banner (30 seconds)

**The banner tells you everything:**

### If it says "MAKING MONEY" (GREEN)
✅ You're profitable
✅ Keep going
✅ Go to Step 3 and scale

### If it says "LOSING MONEY" (RED)
⚠️ Spent more than you made
⚠️ This is NORMAL in testing phase (days 1-3)
⚠️ If still red on day 7, stop and evaluate

### If it says "TESTING" (BLUE)
🔵 Still collecting data
🔵 Wait for 20+ clicks per offer
🔵 Check back tomorrow

---

## Step 3: Make ONE Decision (3 minutes)

### Decision Tree:

**Q: Is the banner GREEN?**

**YES → Click "Scale Winners" button**
- System automatically finds ALL offers with >2% conversion
- Splits remaining budget across them proportionally
- More money goes to best performers
- You don't have to pick - system does it for you

**NO → Do nothing**
- Let it keep testing
- Check back tomorrow
- If red for 7+ days, click "Stop All"

## Weekly Summary (Every 7 Days)

Once per week, check these three numbers:

```
Money In:  $XXX  (green number)
Money Out: $XXX  (red number)
Profit:    $XXX  (blue/green number)
```

**If Profit is positive:**
- You're winning
- Keep scaling winners
- Reinvest profits if you want faster growth

**If Profit is negative after 7 days:**
- Testing didn't find winners yet
- Either: stop and save remaining budget
- Or: try 3 more days with different offers

---

## Emergency Stop

**Click "Stop All" if:**
1. Spent >$400 with $0 revenue
2. All offers showing <1% conversion after 100 clicks each
3. You need to pause for any reason

**This immediately stops:**
- All traffic buying
- All budget spending
- All automated posting

**Your data is saved** - you can restart later.

## First Week Expectations

### Days 1-2: Testing Phase
- Banner: BLUE ("Testing")
- Money Out: ~$100
- Money In: $0-$200
- Action: Do nothing, let it test

### Days 3-5: Optimization Phase
- Banner: Might turn GREEN
- Money Out: ~$200
- Money In: $200-$800
- Action: If green, click "Scale Winners"

### Days 6-7: Early Scaling
- Banner: Should be GREEN if working
- Money Out: ~$300
- Money In: $500-$2,000
- Action: Keep scaling winners

### Day 13: Final Results
- Banner: GREEN or RED - you'll know
- If GREEN: Total profit $500-$10,000
- If RED: Stop, save remaining budget, try different offers
